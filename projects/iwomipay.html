IWOMIPAY Payment Gateway API

  • Tech Stack: Java Spring, React js, HTML, CSS, Bootstrap, Php, wordpress, REST API, mysql
  • Website URL: Website Link

A secure cash management solution, allowing companies to accept payments from their invoices online, and instantly credited to their bank accounts. The platform has all the means of payment available in the CEMAC ecosystem (Mobile Money, GIMAC, Visa, MasterCard, PayPal, etc.). Whatever the means of payment used by the customer, the biller receives the credit for the payment in his account at the bank in real time.

Role:

Integration and aggregation of partner APIs -> MTN, Orange Money API, Paypal API, Cybersource (VISA) API for card payments, Smobilpay API for Bill payments and Airtime topUp Design, development of solution and new features, setup, deployment and configuration of solution, Product Manager, Technical lead, head of integration.
